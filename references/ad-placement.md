## 1. 投流的本质——用钱换注意力

**底层公式**：
```
投放总收益 = Σ (曝光量 × eCTR × eCVR × 客单价) - 投放总花费
           = Σ (Impression_i × p(Click_i) × p(Conversion_i) × LTV_i) - Budget
```

投流优化的本质：**在预算约束下最大化这个公式的输出**。

## 2. 广告交易机制全景

### 2.1 四条交易路径

```
品牌方（广告主）
    │
    ├──[1]→ DSP（需求方平台）→ Ad Exchange → SSP（供给方平台）→ 媒体
    │      程序化竞价，机器对机器，毫秒级
    │
    ├──[2]→ 平台广告系统（巨量引擎/腾讯广告/阿里妈妈/Google Ads）
    │      入驻平台，在平台内部竞价，最主流
    │
    ├──[3]→ KOL/KOC 采买（星图/蒲公英/微任务 或 私下合作）
    │      直接采买创作者内容，非标准化
    │
    └──[4]→ 传统采买（签合同→排期→执行）
           电视/户外/OTV贴片/节目赞助，非竞价
```

### 2.2 程序化广告的四种交易模式

| 模式 | 机制 | 价格 | 保量 | 适用 |
|------|------|:--:|:--:|------|
| **RTB 公开竞价** | 每次曝光实时竞价，价高者得 | 浮动 | ✗ | 效果广告，追求 ROI |
| **PMP 私有市场** | 在限定买家池内竞价 | 中高 | ✗ | 品牌广告，需要媒体质量 |
| **Preferred Deal** | 固定价格，优先购买权 | 中 | 半保 | 需要确定性 + 灵活性 |
| **Programmatic Guaranteed** | 固定价格+保量，程序化执行 | 高 | ✓ | 品牌大曝光 |

**交易效率 vs 媒体质量**：

```
效率高                          质量高
 ← RTB ── PMP ── Preferred Deal ── PG →
```

## 3. 竞价广告核心机制

### 3.1 eCPM——决定谁能曝光的唯一指标

```
eCPM = 出价(Bid) × 预估点击率(eCTR) × 预估转化率(eCVR) × 1000
```

**这意味着**：
- 出价高≠能赢。出价低但 eCTR/eCVR 高（素材好、产品好）一样能赢
- **素材质量是隐形的「出价加成」**——好素材让系统预估你的 CTR 更高，相当于以更低出价获得更多曝光
- 平台的「AI 优化」本质上是在帮你调整 Bid，但 eCTR/eCVR 取决于你自己的素材和产品

### 3.2 竞价广告的「二阶拍卖」

广义二阶价格：你只需支付刚好能击败下一名的价格，而非你的全部出价。

```
广告位有 3 个竞争者：
A 出价 10 元（eCPM 最高）→ 胜出，实际支付 ≈ 7 元（击败B所需的最低价格）
B 出价 7 元  → 第二，不展示
C 出价 3 元  → 第三，不展示
```

但现代平台（巨量/Google）并非纯二阶拍卖，而是混合 GSP + 质量分 + AI 预估的复杂计算。

## 4. 主流投放渠道与指标

### 4.1 中国主流平台投流渠道

| 平台 | 主力投放形式 | 典型指标基准 | 适合 |
|------|------------|-------------|------|
| **巨量引擎（抖音）** | 信息流短视频+直播投流 | CPA 5-50元（品类差距大） | 全品类，效果+品牌 |
| **腾讯广告** | 朋友圈+公众号+视频号 | CPM 30-80元 | 品牌曝光+私域 |
| **阿里妈妈** | 搜索+推荐+万相台 | ROI 驱动 | 电商导向 |
| **小红书** | 信息流+搜索广告 | CPE（互动成本）2-10元 | 种草+品牌建设+转化 |
| **微博** | 粉丝通+热搜 | CPM 5-15元 | 话题引爆+娱乐营销 |
| **B站** | 信息流+UP主商业合作 | CPA 偏高 | 年轻人群+深度内容 |
| **快手** | 信息流+直播 | 下沉市场 | 性价比获客 |
| **百度** | 搜索竞价+信息流 | CPC 2-20元 | 主动搜索需求捕获 |

### 4.2 关键指标全解

| 指标 | 公式 | 含义 | 诊断 |
|------|------|------|------|
| **Impression** | — | 曝光次数 | 量够不够？ |
| **CTR** | 点击/曝光 | 素材吸引力 | 低→素材或人群问题 |
| **CPM** | (花费/曝光)×1000 | 千次曝光成本 | 反映竞争激烈度和受众价值 |
| **CPC** | 花费/点击 | 每次点击成本 | CPM÷(CTR×1000) |
| **CVR** | 转化/点击 | 落地页转化效率 | 低→落地页/产品问题 |
| **CPA/CAC** | 花费/转化 | 获得一个客户/转化的成本 | 核心考核指标 |
| **ROAS** | 收入/广告花费 | 广告支出回报（不含货品成本） | 如 ROAS=3 → 花1元赚3元GMV |
| **ROI** | (收入-成本)/成本 | 全成本投资回报（含货品） | ROAS>ROI |
| **LTV** | Σ 客户生命周期价值 | 长期价值 | CAC 必须 < LTV |
| **频次** | 曝光/独立用户 | 人均看到几次 | >7 可能造成厌烦 |

## 5. 出价策略

### 5.1 策略速查

| 出价策略 | 优化目标 | 适用阶段 | 优缺点 |
|----------|----------|----------|--------|
| **CPM 手动** | 曝光量 | 需要保量时 | 量可控，成本可能浪费 |
| **oCPM** | 曝光+转化 | 追求跑量同时兼顾转化 | 平衡型，最常用 |
| **oCPC** | 点击+转化 | 流量优先+转化兼顾 | 适合需要流量的场景 |
| **oCPA** | 严格控制转化成本 | ROI 敏感型投放 | 深度转化最准，但冷启动慢 |
| **最大转化量** | 在预算内最大化转化数 | 放量期/跑量冲刺 | 量最大化，成本可能失控 |
| **ROI 出价** | 稳定 ROI 水平 | 成熟期+稳定产品 | 最智能但需要足够数据 |

### 5.2 冷启动方法论

新计划上线前 3 天是「学习期」。系统在探索什么样的人对你的广告有反应。

**冷启动做对三件事**：
1. **出价给足**：学习期出价 = 目标出价 × 1.3-1.5，给系统探索空间
2. **预算不卡**：学习期内不要频繁调预算，让系统稳定学习
3. **耐心给够**：至少跑 3 天 + 50 个转化后再做判断，不要第 1 天数据差就关

**冷启动失败信号**：
- 48 小时消耗 < 预算的 20%
- 3 天转化数 < 20
- CPM 持续上升 + 转化下降

## 6. 投流账户结构

### 6.1 标准层级

```
Campaign（广告组/推广计划）
  └── Ad Set（广告组/单元）  ← 在这里设：预算、出价、人群定向、投放时段
        └── Ad（广告/创意）   ← 在这里设：素材、文案、落地页
```

### 6.2 结构最佳实践

```
一个 Campaign 下：
  ├── 2-3 个 Ad Set（不同人群包 + 相同素材测试人群）
  │   每个 Ad Set 下：
  │    └── 3-5 个 Ad（不同素材，竞争出最好的）
  │
  ├── 1-2 个 Ad Set（不同出价策略测试）
  │
  └── 总量控制：一个账户同时在线计划 ≤ 20 个活跃计划
```

## 7. A/B 测试——投流优化的唯一科学方法

### 7.1 五要素

| 测试变量 | 示例 | 建议测试周期 |
|----------|------|:--:|
| 素材（最重要） | 口播 vs 剧情 vs 产品展示 | 3-7 天 |
| 文案 | 痛点开头 vs 场景开头 vs 价格开头 | 3-5 天 |
| 人群定向 | 竞品人群 vs 品类兴趣 vs 宽泛 | 5-7 天 |
| 落地页 | 产品详情 vs 活动页 vs 测评页 | 5-7 天 |
| 出价策略 | oCPM vs oCPA | 7 天+ |

### 7.2 A/B 测试纪律

- **一次只测一个变量**——你改素材又改人群，无法归因
- **样本量要够**——每个版本至少 1000 次曝光或 30 个转化再判断
- **置信度检查**——数据差异 < 10% 可能是随机波动，重复测试
- **胜出标准**：核心指标（CPA/ROAS）差异 > 15% 且样本量达标 → 关败留胜

## 8. 投流优化循环

### 8.1 永远在转的 PDCA

```
    1. 设定 KPI (目标 CPA / ROAS / 量级)
              ↓
    2. 搭建 / 新建计划
              ↓
    3. 数据回收（至少 3 天）
              ↓
    4. 诊断决策 ──────┐
       ├── 表现好 → 放量（+20% 预算/天）→ 回到 3
       ├── 表现差 → 关停 → 回到 2
       └── 表现不稳定 → 不改 → 回到 3 继续观察
```

### 8.2 诊断框架：当投放数据不好

**顺着漏斗往上排查**：

```
问题是什么指标？
├── 曝光低 → 出价太低？人群太窄？素材已被系统判定低质？
├── CTR 低 → 素材不吸引人？人群不匹配？文案问题？
├── CVR 低 → 落地页问题？产品价格/卖点问题？人群意向问题？
├── CPA 高 → 转化链路太长？竞争太激烈（出价战）？
├── ROAS 低 → 客单价太低？复购率差？退货率高？
└── 消耗快但无转化 → 大概率刷量/低质流量，检查点击-转化时间差
```

### 9.1 按品牌阶段

| 品牌阶段 | 品牌广告 | 效果广告 | 内容/KOL | 实验预算 |
|:--|:--:|:--:|:--:|:--:|

| 成长期 | 30% | 40% | 20% | 10% |
| 成熟期 | 20% | 55% | 15% | 10% |
| 防守期 | 15% | 60% | 15% | 10% |

### 9.2 70-20-10 法则

```
70% → 已验证有效的渠道/策略
20% → 已验证渠道的新尝试
10% → 完全实验性的渠道/策略
```

## 10. 投流禁忌

1. **频繁调价**——一天调价 > 3 次，系统学习期不断重置，计划永远冷启动
2. **预算时开时关**——今天开明天关，系统无法建立稳定模型
3. **素材疲劳不换**——同一素材跑 2 周以上，频次 > 7，用户已经厌烦
4. **定向过窄**——人群包太小（< 10 万），系统无法充分探索
5. **同时开太多计划**——内部竞争，自己和自己竞价推高成本
6. **不看归因窗口**——点击后 1 天转化 vs 7 天转化 vs 30 天转化，数据差异巨大
7. **不区分新老客**——新客获客成本通常是老客的 3-5 倍，不分开考核会误判

## Earned Media: 媒体与 KOL 关系管理

投流是 paid——花钱买曝光。Earned media 是不直接花钱买、而是通过关系和信息获得的曝光。酒水行业 earned media 的核心资产是：酒评人、侍酒师、行业媒体、生活方式媒体、以及自发分享的消费者。

### 媒体/KOL 联系人分级管理

| 级别 | 定义 | 数量 | 维护频率 | 维护方式 |
|:--:|------|:--:|:--:|------|
| **A 级** | 行业顶级酒评人/头部媒体/头部 KOL | 5-10 | 每月 | 专属体验（私享品鉴、一对一沟通）、新品首发权 |
| **B 级** | 活跃媒体/中腰部 KOL/侍酒师 | 20-50 | 每季度 | 定期送样 + 品牌资讯 + 活动邀请 |
| **C 级** | 长尾自媒体/小 KOC/爱好者 | 100+ | 按需 | 新品寄样 + 线上互动 + UGC 激励 |

### 媒体 Pitch 策略

让一个记者/博主对你的酒感兴趣的 4 个钩子：

| 钩子类型 | 适用场景 | 示例 |
|----------|----------|------|
| **独家/首发** | 有新闻价值的事件 | 「我们下个月发布一款全新品类，给你首发」 |
| **数据/趋势** | 有行业洞察的品牌 | 「我们有一组关于年轻人饮酒习惯的数据，可以做一篇趋势报道」 |
| **人物故事** | 有魅力的创始人/酿酒师 | 「我们酿酒师的故事——从金融转行酿酒，你可能想聊聊」 |
| **差异化体验** | 独特产品/工艺 | 「邀请你来我们云南的原料产地，看真实水果是怎么入酒的」 |

### 媒体样品寄送 SOP

```
1. 筛选名单：A级+B级联系人
2. 提前沟通：微信/邮件告知「近期会寄一份新品，希望听到你的真实反馈」
3. 寄送包装：酒 + 品牌 Fact Sheet（一页纸关键信息）+ 品鉴笔记 + 手写卡片
4. 寄后 3 天跟进：「收到了吗？有什么初步印象？」
5. 不做硬性发稿要求——让他们自发决定是否写
6. 如果发了：感谢 + 转发 + 建立长期关系
7. 如果没发：不追问、不施压——下次再寄
```

### 媒体独家/首发策略

| 策略 | 适用 | 注意事项 |
|------|------|----------|
| 独家给 A 级媒体 | 重大新品/战略发布 | 确保独家对象有足够影响力值得你放弃其他媒体 |
| 首发给行业媒体 → 大众媒体跟进 | 行业事件 | 行业媒体先做专业解读 → 大众媒体转载扩散 |
| 群发给媒体通稿 | 常规新闻 | 通稿要真材实料，而非广告软文 |

## Owned Media: 品牌自有内容渠道

品牌自己拥有的传播资产，不依赖平台算法和付费。

| 渠道 | 内容类型 | 运营重点 |
|------|----------|----------|
| 品牌公众号 | 深度内容、品牌故事、行业洞察 | 保持更新频率、内容有独立价值而非广告 |
| 视频号/抖音品牌号 | 短视频、直播、纪录片 | 人格化——不是品牌在说话，是人在说话 |
| 品牌官网/小程序的「媒体中心」 | 新闻稿存档、品牌 Fact Sheet、高清素材下载 | 降低媒体报道门槛——媒体需要的素材一键获取 |
| 品牌 Newsletter | 新品、品鉴笔记、行业观点 | 对核心用户/媒体/KOL 的定期深度沟通 |
| 品牌播客 | 酿酒师访谈、行业对谈 | 建立思想领导力 |
