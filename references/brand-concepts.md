# Brand Concepts — 品牌概念知识库

> 加载时机：当需要深究品牌定义、品牌识别、品牌定位、品牌资产、品牌激活等概念时。

## 1. 品牌本质定义

### AMA 经典定义
品牌是一个「名称、术语、设计、符号或任何其他特征，用于将一个卖方的商品或服务与其他卖方的区分开来」。

### 品牌金字塔（Brand Pyramid）
```
        ┌──────────┐
        │ 灵魂/本质 │  ← Brand Essence（品牌精髓：一句话概括你是谁）
       ┌┴──────────┴┐
       │  品牌个性   │  ← 如果品牌是一个人，TA 什么性格？
      ┌┴────────────┴┐
      │  情感利益     │  ← 消费者感受到什么？（自信、归属、安全感…）
     ┌┴──────────────┴┐
     │  功能利益       │  ← 产品/服务解决了什么具体问题？
    ┌┴────────────────┴┐
    │  属性/特征        │  ← 产品客观属性（原料、工艺、价格…）
    └──────────────────┘
```

### 品牌-产品光谱
```
纯产品 ←——————————————————————→ 纯品牌
   │                                  │
  大米、矿泉水                  Apple、Nike、Hermès
  （功能性购买）                （符号性购买）
```
酒水行业典型位于中偏右——功能（口感）+ 符号（身份/场景）双重属性。

## 2. 品牌识别系统

### 2.1 Kapferer 品牌识别棱镜（6 维度）

```
        外部化（品牌呈现给世界的样子）
        ┌─────────────────┐
  Physique│                 │Personality
  （外形）│    品  牌       │（个性）
        │    识  别       │
Relationship│   棱  镜      │Culture
  （关系）│                 │（文化）
        └─────────────────┘
  Reflection│               │Self-image
  （反射）  │                 │（自我形象）
        内部化（消费者内心世界）
```

**6 维度解析**：

| 维度 | 含义 | 酒水品牌示例提问 |
|------|------|-----------------|
| Physique 外形 | 品牌可感知的物理特征组合 | 瓶型、酒标、颜色、材质 |
| Personality 个性 | 如果品牌是人，什么性格 | 沉稳内敛？张扬热烈？ |
| Culture 文化 | 品牌背后的价值观体系 | 产区文化、酿造传统、品牌故事 |
| Relationship 关系 | 品牌和消费者的关系模式 | 导师？知己？派对伙伴？ |
| Reflection 反射 | 品牌投射的目标消费者画像 | 「喝这酒的人」是谁 |
| Self-image 自我形象 | 消费者通过品牌看到的自己 | 「喝这酒，我是什么样的人」 |

### 2.2 Aaker 品牌识别模型

品牌识别 = 核心识别 (Core Identity) + 延伸识别 (Extended Identity)。

- **核心识别**：品牌的永恒本质，不随产品/市场变化。如 Nike 的「真实的运动表现」
- **延伸识别**：为品牌增加质感和完整性的元素。如品牌口号、品牌故事、品牌符号

同时包含 4 个视角：
1. **品牌即产品**（产品范围、属性、质量/价值、使用场景、使用者、原产国）
2. **品牌即组织**（组织属性、本地 vs 全球）
3. **品牌即人**（品牌个性、品牌-顾客关系）
4. **品牌即符号**（视觉形象、品牌传承）

### 2.3 品牌原型 12 种（Jung 原型 + Mark & Pearson）

| 原型 | 核心渴望 | 品牌举例 | 酒水适配 |
|------|----------|----------|:--:|
| 天真者 | 体验天堂 | 可口可乐、迪士尼 | 果酒/预调酒 |
| 探险家 | 自由探索 | Jeep、Patagonia | 精酿/户外场景酒 |
| 智者 | 追求真理 | Google、BBC | 高端红酒/威士忌知识营销 |
| 英雄 | 证明价值 | Nike、FedEx | 烈酒/能量型 |
| 亡命之徒 | 打破规则 | Harley-Davidson | 限量/叛逆调性 |
| 魔术师 | 实现转变 | Disney、Dyson | 调酒/仪式感 |
| 凡夫俗子 | 归属连接 | IKEA、啤酒 | 大众酒/佐餐酒 |
| 情人 | 获得亲密 | Chanel、Godiva | 高端红酒/香槟 |
| 弄臣 | 享受当下 | M&M's、Old Spice | 派对酒/低度酒 |
| 照顾者 | 保护他人 | 强生、雀巢 | 低度健康酒 |
| 创造者 | 创造永恒 | LEGO、Apple | 工艺酒/限量版 |
| 统治者 | 掌控秩序 | 劳力士、奔驰 | 顶级年份酒/收藏级 |

## 3. 品牌定位

### 3.1 定位的基础

定位不是你对产品做的事，而是**你对消费者心智做的事**（Ries & Trout）。定位 = 在消费者心智中占据一个独一无二的位置。

### 3.2 定位陈述公式

```
对于 [目标人群]，
[品牌名] 是 [品类] 中
能够提供 [核心差异化利益] 的品牌，
因为 [RTB——相信的理由]。
```

**RTB 类型**：
- 产品/成分 RTB：如「12 年单一麦芽」
- 工艺 RTB：如「古法酿造」
- 产地 RTB：如「茅台镇 7.5 平方公里核心产区」
- 专家/权威 RTB：如「IWSC 金奖」
- 消费者 RTB：如「100 万酒友的选择」
- 数据 RTB：如「全渠道销量第一」

### 3.3 品牌架构 4 型

| 架构类型 | 特点 | 酒水行业案例 |
|----------|------|-------------|
| **Branded House** | 所有产品共用主品牌 | 江小白（果立方、梅见早期） |
| **House of Brands** | 多个独立品牌，消费者不知母体 | 百威英博（百威、科罗娜、福佳白互不关联） |
| **Endorsed Brand** | 子品牌独立 + 母品牌背书 | 茅台 × 茅台王子酒 |
| **Hybrid** | 以上混合 | 保乐力加（部分品牌独立，部分背靠集团） |

### 3.4 品牌重新定位策略

何时需要重新定位：
- 市场变化（品类萎缩/新进入者改变格局）
- 消费者变化（代际迁移导致原定位失效）
- 品牌老化（增长停滞，认知固化）
- 竞争挤压（竞品占据了你的定位空间）

重新定位的风险：老用户流失 > 新用户获取。

## 4. 品牌资产 (Brand Equity)

### 4.1 Aaker 品牌资产五星模型

```
        ┌─────────────┐
        │  品牌忠诚度   │ ← 降低营销成本、贸易杠杆、吸引新客、应对竞争
        ├─────────────┤
        │  品牌知名度   │ ← 锚定联想、熟悉-喜欢、实力信号
        ├─────────────┤
        │  感知质量     │ ← 购买理由、差异化定位、溢价基础
        ├─────────────┤
        │  品牌联想     │ ← 帮助信息处理、差异化、购买理由、正面态度
        ├─────────────┤
        │ 其他品牌资产   │ ← 专利、商标、渠道关系
        └─────────────┘
              ↓
         为顾客创造价值 ← → 为企业创造价值
```

### 4.2 Keller CBBE 金字塔（Customer-Based Brand Equity）

```
            ┌──────────────┐
            │   共鸣 Resonance  │  ← 行为忠诚、态度依附、社群感、主动参与
           ┌┴──────────────┴┐
           │  评判 + 感受     │  ← 质量、可信度、优越性 / 温暖、有趣、安全
          ┌┴────────────────┴┐
          │   表现 + 形象      │  ← 首要/次要特征、可靠性、风格、价格
         ┌┴──────────────────┴┐
         │     显著性 Salience  │  ← 深度广度知名度、品类识别
         └────────────────────┘
```

**建塔逻辑**：显著性（我认识你）→ 表现+形象（你是什么）→ 评判+感受（我喜欢你吗）→ 共鸣（我们是一起的）。必须从下往上建，不可跳步。

### 4.3 品牌资产评估方法

| 方法 | 说明 | 输出 |
|------|------|------|
| **Interbrand 法** | 财务预测 × 品牌角色 × 品牌强度 | 货币化品牌价值 |
| **品牌资产十项指标** (Aaker) | 10 个忠诚度+品质+联想+知名度指标 | 综合评分卡 |
| **价格溢价法** | 品牌产品 vs 无品牌/竞品的溢价 | 百分比溢价 |
| **联合分析法** | 消费者在不同属性组合中的选择 | 品牌相对效用值 |

## 5. 品牌激活

### 品牌年轻化 vs 品牌复兴

| | 品牌年轻化 | 品牌复兴 |
|------|------|------|
| 品牌状态 | 当前健康，担忧未来老化 | 已明显下滑 |
| 核心策略 | 微调产品/形象/沟通 | 回到品牌根源+彻底改造 |
| 风险 | 过度反应失去核心用户 | 改动过大导致不可逆伤害 |
| 案例 | 茅台×瑞幸酱香拿铁 | Gucci 的 Alessandro Michele 时代 |

### 品牌延伸评估

扩展品牌到新品类时，评估 3 个维度：
1. **品类契合度**：新品类和原品牌的自然关联？
2. **品牌资产可迁移性**：原品牌的核心联想能否赋能新品？
3. **品牌稀释风险**：失败是否会伤害主品牌？

## 附录：品牌级场景锁定与教育策略

以下工具源自品类创造框架（`references/category-creation.md`），但同样适用于**已有品类中的品牌定位**。品类创造和品类竞争不是两套工具，是同一套工具在两个层级上的应用。

### 跨层级对照

| 工具 | 品类创造层级 | 品牌竞争层级（本附录） |
|------|------------|---------------------|
| **场景锁定** | 定义「这个新品类什么时候用」 | 定义「这个品类中，什么场景下选我而非竞品」 |
| **教育节奏** | 教育「这个新东西是什么」→ 品类认知 | 教育「这个品牌为什么不一样」→ 品牌差异 |
| **种子用户** | 品类最早的尝鲜者 | 品牌在品类内的早期拥护者——对现有选项不满意的人 |

### A. 场景锁定

已有品类 ≠ 已有场景。品类里的每个品牌，必须**占领一个明确的细分场景**。谁先占领，谁就拥有该场景的心智。

```
品类  →  场景  →  品牌

白酒  →  商务宴请（最重要时刻） →  茅台
     →  朋友小聚（微醺不醉）   →  江小白
     →  家庭日常（性价比）     →  牛栏山
     →  送礼（面子）           →  五粮液/国窖

啤酒  →  看球聚会（热闹）       →  雪花/青岛
     →  品质独享（口感）       →  精酿品牌
     →  女性微醺（轻松）       →  果味啤酒
```

**场景锁定公式**：
```
在 [品类] 中，当 [具体场景] 时，[品牌名] 是首选。
不是因为 [品牌] 比竞品好在哪，而是因为 [场景 × 品牌] 的组合是唯一的。
```

**场景检查**：
- [ ] 这个场景真实存在吗？（不是你自己想象的）
- [ ] 这个场景目前在用什么方案？（竞品？替代品？）
- [ ] 这个场景下，消费者选择时的第一排序依据是什么？
- [ ] 竞品能一句话占领这个场景吗？如果能，你凭什么抢？

### B. 教育节奏（品牌版）

适用于品类已存在、但需要教育「本品牌为什么不一样」的场景。

```
第 1 层：认知（一句话记住你——不是品类，是差异）
  → 「[品牌名]，[品类] 里 [最 X] 的那个」
  → 例：「江小白，白酒里最懂年轻人的那个」

第 2 层：理解（为什么不一样）
  → 回答「你凭什么这么说」
  → RTB：产品、工艺、原料、数据

第 3 层：信服（为什么值得）
  → 社会证明：谁在喝？什么场景？什么评价？
  → 试用/体验：降低第一次尝试门槛
  → 对比：和现有选项的具体差异

第 4 层：传播（让用户帮你说话）
  → 用户转述的一句话，必须你设计好，不能指望用户自己想
  → 「你喝的这是什么？」——这个答案要能 3 秒说完
```

**品牌教育 vs 品类教育的差异**：

| 层级 | 品类创造（Module 7） | 品牌竞争（这里） |
|:--:|------|------|
| 第 1 层 | 「这是一种新的喝法」 | 「这个品牌不一样在哪」 |
| 第 2 层 | 「为什么和所有已有酒不同」 | 「为什么和同品类其他品牌不同」 |
| 第 3 层 | 「为什么值得尝试一个新品类」 | 「为什么值得从竞品切换过来」 |
| 第 4 层 | 「帮你跟朋友解释你喝的这是什么」 | 「帮你跟朋友解释你为什么选这个品牌」 |

### C. 种子用户选择

即使品类成熟，新品牌的第一批用户也不该是大众。他们是**对现有选项不满意的人**。

**种子用户筛选四问**：
1. 谁正在使用现有方案（竞品/替代品），但**有明显不满**？
2. 谁的需求和现有品类的供给之间存在**最大的缺口**？
3. 谁最容易触达（地理/渠道/社群）？
4. 谁尝试后**最有动力分享**？

**种子用户的传播设计**：
```
让 TA 喝完以后，给朋友发一条消息：

「我最近发现一个 [品类] 叫 [品牌名]，
 它和 [现有认知中的品类代表] 不一样，
 它是 [一句话差异]。
 你可以试试。」
```

**检查**：这段话能不能从你的种子用户嘴里自然说出来？如果不能——你的教育节奏第 4 层没设计好。

## 附录 B：高管可见度与思想领导力

酒水品牌的核心资产之一是**创始人/酿酒师/品牌主理人**的个人 IP。PR 的职责是把 TA 的专业性和故事推到行业和公众面前。

### 高管可见度矩阵

| 平台 | 内容类型 | 频率 | 目的 |
|------|----------|:--:|------|
| 行业论坛/峰会 | 主题演讲、圆桌嘉宾 | 2-3次/年 | 行业话语权——「这个品类的话题绕不开 TA」 |
| 媒体专访 | 深度人物专访、行业观点 | 3-5次/年 | 个人品牌建设 |
| 品牌自有内容 | 公众号专栏、播客、视频 | 月更 | 常态化思想输出 |
| 社交媒体 | LinkedIn/微信个人号内容 | 周更 | 降低距离感 |
| 评奖/榜单 | 「40 under 40」等行业榜单 | 按需 | 第三方背书 |

### 个人叙事框架

**三条叙事线，不是三条广告语**：

| 叙事线 | 适合的人物类型 | 核心问题 |
|--------|--------------|----------|
| **创业故事** | 转型/跨界创始人 | 「你为什么放弃 X 来做酒？」 |
| **匠心故事** | 酿酒师/技术型创始人 | 「你花了多长时间只做一件事？」 |
| **行业洞见** | 行业老兵/趋势判断者 | 「你觉得这个行业最大的机会/威胁是什么？」 |

### 媒体采访准备原则

- 准备 3 个**必须说出去**的核心信息——不管记者问什么，找机会说出去
- 准备 1 个**绝对不能说的话**——把禁区明确写下来
- 采访不是考试——不需要回答所有问题。不会的就说「这个我回去确认一下回复你」
- 没有「随便聊聊」的采访——所有录音都是公开的，所有对话都可能被引用

## 附录 C：定价策略

定价不是成本+利润，是品牌定位的数学表达。酒水的定价是一个**信号系统**——每一个价格数字都在向不同的人传递不同的信号。

### 定价的三层信号

| 价格层 | 向谁发信号 | 信号内容 | 定错了的结果 |
|--------|-----------|----------|------------|
| **出厂价** | 经销商 | 「这个品牌值多少钱、利润空间多大」 | 太高→无人代理；太低→被视为低端 |
| **终端价** | 消费者 | 「我在市场中处于什么位置」 | 和品牌定位不符→消费者困惑 |
| **电商价** | 经销商+消费者 | 「线上是什么态度」 | 低于终端价→经销商集体抗议 |

### 价格带选择

酒水定价不是线性思考，是**价格带卡位**。选价格带 = 选战场。

| 价格带（白酒示例） | 人群特征 | 决策逻辑 | 品牌门槛 |
|:--|------|------|:--|
| 超高端 (>800) | 送礼/收藏/面子消费 | 品牌 > 价格 | 需要历史/稀缺性/产区背书 |
| 次高端 (300-800) | 商务宴请/品质自饮 | 品牌+品质双重考量 | 需要品牌故事+获奖+口碑 |
| 中端 (100-300) | 日常宴请/朋友聚会 | 性价比+口感 | 需要区域渗透+渠道密度 |
| 大众 (<100) | 日常自饮/高频消费 | 价格+可得性 | 需要供应链效率+渠道规模 |

**价格带迁移的黄金法则**：向上迁移需要**三代人的时间**或**一次品类革命**。向下迁移只需要一纸调价通知，且不可逆。

### 渠道价差设计

```
出厂价 ──→ 一批价 ──→ 二批价 ──→ 终端价 ──→ 消费者支付

价差1 (品牌→一批)  价差2 (一批→二批)  价差3 (二批→终端)
```

| 价差 | 典型范围 | 作用 | 常见问题 |
|------|:--:|------|------|
| 价差1 | 20-35% | 一级商利润+仓储物流服务 | 太小→一级商无动力；太大→品牌利润被压缩 |
| 价差2 | 10-15% | 二级商分销利润 | 和一级差距太小→二级直接找一级绕过 |
| 价差3 | 20-40% | 终端零售利润 | 即饮终端溢价高(30-40%)；非即饮低(20-25%) |

### 涨价/降价策略

**涨价的前提条件（缺一不可）**：
1. 品牌在价格带站稳脚跟——消费者认可当前位置
2. 有覆盖涨价的「故事」——原料升级、年份升级、产区限定
3. 竞品没有空档等着你退出后的位置
4. 经销商库存处于低位——否则他们借涨价清库存，价格体系崩溃

**降价的唯一安全方式**：不降主品牌标价，而是推出「新规格/新系列」在更低价格带竞争。

### 区域价格管理

同一瓶酒在不同区域的价格容忍度不同。品牌方要管的是**价差不超过 15%**——超过这个阈值：
- 窜货开始发生（经销商从低价区进货卖到高价区）
- 高价区经销商积极性崩溃
- 终端价格混乱→消费者信任消失

### 线上线下价格冲突

线上不能比线下便宜。线上便宜一块钱 = 线下每一瓶都「贵了」。解决方案：
- 线上卖**不同规格**（小瓶装/礼盒装/限定款）→ 不可比价
- 线上卖**不同体验**（定制酒标/品鉴教程捆绑）→ 不只是价格竞争

### 定价检查清单

- [ ] 终端价是否和品牌定位（原型/价格带）一致？
- [ ] 每一层渠道是否有合理利润（一级 ≥ 20%、终端 ≥ 20%）？
- [ ] 最高价区和最低价区差距是否 < 15%？
- [ ] 线上价格是否 ≥ 线下终端价？
- [ ] 涨/降价之前是否检查了经销商库存水平？

## 附录 D：品牌使命、愿景、价值观

品牌定位解决「我们是谁、卖给谁」。使命愿景价值观解决「我们为什么存在、要去哪、怎么做事」——这是品牌文化的内核，也是 Brand Book 的第一章。

### 使命 (Mission)

品牌为什么存在——除了赚钱之外。

**使命公式**：
```
[品牌名] 的存在是为了 [让谁] [实现什么改变/解决什么问题]。
```

- ✓ 「让每一个疲惫的成年人，在一天结束时有 15 分钟属于自己的放松时刻」
- ✗ 「致力于提供卓越的酒精饮料产品与服务」——空洞，没有人记得住

**检查**：员工能不看稿子、用自己的话说出品牌的使命吗？如果不能——使命太长或太虚。

### 愿景 (Vision)

品牌在可预见的未来要成为什么。

**愿景公式**：
```
在 [时间范围] 内，[品牌名] 将成为 [什么品类/市场] 中 [什么地位]。
```

- ✓ 「5 年内，成为年轻人睡前放松的第一选择」
- ✗ 「成为行业领先的综合性酒水品牌」——无时间、无边界、无差异化

### 价值观 (Values)

不是贴在墙上的标语——是在两难决策时，品牌选哪边的判断标准。

**价值观设计原则**：
- 3-5 条，每条一个词或一句话
- 每条可以展开为一个具体场景：「当 [两难] 时，我们选 [A] 不选 [B]」
- 每条有反面——如果你说不出「什么情况下这个价值观会被打破」，说明这不是真的价值观

**示例**：
```
价值观：自然是第一配方
解读：能用天然的，绝不用合成的。哪怕成本更高、保质期更短。
反面：如果为了口味稳定性加入人工香精，就打破了这个价值观。
```

## 附录 E：品牌语调（Tone of Voice）

品牌语调 = 品牌说话的「语气」。不是说什么（那是定位和核心信息解决的问题），是**怎么说**。

### 语调定义四维模型

| 维度 | 我们 | 不是 | 说明 |
|------|------|------|------|
| 正式 ↔ 轻松 | [选一边] | [反面] | 「您好，欢迎品鉴我们的产品」vs 「嘿，试试这个」 |
| 专业 ↔ 亲切 | [选一边] | [反面] | 「采用 12987 工艺酿造」vs 「老法子做的，味道不一样」 |
| 大胆 ↔ 谨慎 | [选一边] | [反面] | 「白酒也能一个人喝」vs 「白酒文化源远流长」 |
| 热情 ↔ 克制 | [选一边] | [反面] | 「太好喝了你要试试！」vs 「值得一试」 |

### 语调一致性的两个铁律

1. **不同渠道，语调可以微调，但人格不能变**。微博可以轻松一点、公众号可以深一点——但消费者应该能认出来「这是同一个品牌在说话」。
2. **最高频的一句话决定语调**。消费者在终端听到你的品牌被推荐的那句话——「XX 啊？那个 [一句话] 的」——这句话里的关键词就是你语调的浓缩。

### 语调示例

```
品牌：江小白
- 正式↔轻松：轻松——「我是江小白，生活很简单」
- 专业↔亲切：亲切——不讲工艺讲心情
- 大胆↔谨慎：大胆——在白酒行业说「白酒不等于应酬」
- 热情↔克制：克制——不呐喊，是自言自语式的安静表达

品牌：茅台
- 正式↔轻松：正式——「国酒茅台」
- 专业↔亲切：专业——讲工艺、讲年份、讲稀缺
- 大胆↔谨慎：谨慎——不说错话
- 热情↔克制：克制——不需要激动，地位不需要声张
```
