# Content Generation — 营销内容生成

> 加载时机：完成策略推演后，需要将策略转化为可执行内容（文案、稿件、邀请函等）。

---

## 1. 设计原则

内容生成是策略的最后一公里。不做独立创意，只做**策略驱动的结构化内容生成**。

**三条铁律**：
1. **先生成策略，再生成内容**——没有策略框架的内容 = 随机乱写
2. **标注品牌信息缺口**——生成时明确标注「[品牌名]」「[产品名]」「[价格]」等需替换字段
3. **多版本产出**——任何内容至少出 2-3 个不同角度/调性的版本，供人工选择

---

## 2. 品牌定位内容

### 2.1 定位陈述

**输入**：STP 结论（目标人群 + 差异化利益 + RTB）
**输出**：一句话定位陈述 + 三段式品牌简介

**模板**：

```
【一句话定位】
[品牌名] 是专为 [目标人群] 打造的 [品类]，
以 [核心差异点] 让每一次 [消费场景] 都 [情感利益]。

【品牌简介】
[品牌名] 诞生于 [产地/年份]，[品牌故事 1-2 句]。
我们相信 [品牌信念]。
因此，我们 [做了什么不同的事]，
只为 [目标人群] 在 [核心场景] 时，能 [获得什么]。

【RTB 支撑】
- [证据 1]
- [证据 2]
- [证据 3]
```

### 2.2 品牌故事

**输入**：品牌原型 + 产区/工艺/人物故事素材
**输出**：300-500 字品牌故事

**三种故事框架**：
- **产区故事型**：「在 [产区] 的 [地理特征]，每一滴 [产品] 都承载着 [风土特征]」
- **匠心传承型**：「[人物] 用了 [时间] 只做一件事——[核心工艺]」
- **颠覆创新型**：「当所有人都说 [行业惯例]，我们问了一句 [为什么不能]」

## 3. Campaign 创意内容

### 3.1 广告文案（多版本）

**输入**：Big Idea + 目标人群 + 核心信息 + 情感调性
**输出**：3-5 版广告文案（每版 50-100 字）

**版本策略**：
- V1: 直击痛点型——「[痛点场景]？[品牌] 替你 [解决方案]」
- V2: 场景描绘型——「在 [美好场景] 的时刻，[品牌] 让这一刻更 [情感]」
- V3: 社交货币型——适合被转发/截图的金句式文案
- V4: 理性说服型——「[数据/事实]，所以 [结论]」

### 3.2 社媒帖子

**输入**：Campaign 主题 + 平台（小红书/抖音/微博/朋友圈）
**输出**：适配平台的帖子文案 + 配图建议

**平台适配**：

| 平台 | 文案风格 | 篇幅 | 配图要求 |
|------|----------|------|----------|
| 小红书 | 「真实分享」口吻、种草风 | 100-300字 | 高颜值、生活化 |
| 抖音 | 前 3 秒钩子、口语化 | 口播 30-60 秒 | 竖屏、动态 |
| 微博 | 话题感、可互动 | 140字内 | 信息图/表情包 |
| 朋友圈 | 克制、高级感 | 50字内 | 单张精修图 |

## 4. 活动内容

### 4.1 品鉴会邀请函

**输入**：活动 6W2H + 品牌调性
**输出**：邀请函文案（正式版 + 社交媒体版）

```
【正式版邀请函】

尊敬的 [姓名]：

[品牌名] 诚挚邀请您参加 [活动名称]。

在 [场地描述]，我们将带您：
· 品鉴 [N] 款 [品类]，其中包括 [亮点酒款]
· 聆听 [讲师/嘉宾] 分享 [主题]
· 佐以 [配餐描述]

时间：[日期] [时间]
地点：[地址]
着装要求：[如有]

名额有限，请于 [截止日期] 前回复。

期待与您共度一个 [情感形容词] 的 [时间描述]。

[品牌名] 敬上
```

### 4.2 经销商大会演讲稿框架

**输入**：战略要点 + 新品信息 + 订货政策
**输出**：演讲稿大纲 + 开场/结尾全文

```
【演讲结构】

1. 开场 (5min): 感谢 + 回顾
   「过去一年，我们共同完成了 [关键数据]……」
   
2. 行业趋势 (10min): 宏观 → 品类 → 机会
   「这个行业正在发生 [3个变化]……」
   
3. 品牌战略 (15min): 我们要去哪、为什么能到
   「我们的方向是 [战略核心]，
    因为 [3个理由]……」
   
4. 新品发布 (15min): 产品 → 定位 → 为什么能卖
   「今天带给各位的 [新品]，
    解决了消费者的 [痛点]……」
   
5. 政策发布 (10min): 订货政策 → 激励方案
   「为了支持各位，今年的政策是 [核心条款]……」
   
6. 结尾 (5min): 信心 + 行动号召
   「[一句话总结]。让我们一起 [目标]。谢谢。」
```

### 4.3 品牌之夜活动文案

**输入**：活动主题 + 亮点 + 品牌调性
**输出**：预热海报文案 + 活动后回顾

## 5. 投流/电商内容

### 5.1 信息流广告文案

**输入**：产品卖点 + 目标人群 + 优惠信息
**输出**：3-5 条信息流文案（适配抖音/腾讯）

**信息流文案公式**：
```
[痛点/场景钩子] + [品牌/产品解决方案] + [RTB/优惠] + [行动指令]
```

**示例结构**：
```
钩子类型：
- 「还在 [痛点行为]？试试 [产品]」
- 「[数字] 个人里有 [数字] 个选择了 [产品]」
- 「[场景]，你需要一杯 [产品]」

RTB 展示：
- 「[权威背书/数据]」
- 「[限时优惠]」

CTA：
- 「点击了解」
- 「限时 [N] 折，手慢无」
```

### 5.2 落地页文案结构

```
1. 头部 Banner: 核心卖点一句话 + CTA按钮
2. 痛点共鸣: 「你是否也……」
3. 解决方案: 产品/服务如何解决
4. 产品展示: 关键卖点 × 3（图文并茂）
5. 社会证明: 数据/好评/背书
6. 优惠/紧迫感: 限时/限量
7. 底部 CTA: 再次行动号召
```

## 6. 内容审核清单（发前必查）

- [ ] 品牌名、产品名、价格、日期等关键信息准确？
- [ ] [方括号] 占位符已替换为实际信息？
- [ ] 文案调性匹配品牌人设？
- [ ] 无敏感词汇、无政治隐喻？
- [ ] 适配目标平台的内容格式/篇幅？
- [ ] CTA（行动号召）清晰？
- [ ] 如需配图，已提供配图建议或方向？
