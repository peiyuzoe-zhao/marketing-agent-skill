# Market Insights — 市场洞察

> 加载时机：需要市场规模、消费者画像、竞品分析、行业趋势、政策环境等前置洞察时。
> 核心原则：先搞清楚战场再打仗。不做无洞察的策略。

---

## 1. 洞察框架

### 5C 分析

```
Context（环境） — 宏观环境、政策、技术趋势
  ↓
Category（品类） — 市场规模、增长率、价格带、趋势
  ↓
Consumer（消费者） — 谁在买、什么时候买、为什么买/不买
  ↓
Competitor（竞争） — 直接竞品、替代品、潜在进入者
  ↓
Company（自身） — 品牌现有资产、能力、差距
```

### 数据来源分级

| 级别 | 来源 | 成本 | 可靠性 | 用途 |
|:--:|------|:--:|:--:|------|
| L1 免费公开 | 百度指数/微信指数/行业报告摘要 | 免费 | 中 | 初步判断方向 |
| L2 付费报告 | 欧睿/尼尔森/酒业家报告 | 数千-数万 | 高 | 市场规模与份额测算 |
| L3 一手调研 | 消费者问卷/终端走访/经销商访谈 | 数万-数十万 | 最高 | 最终决策依据 |
| L4 自有数据 | 供应链进销存/价盘/开瓶率（数据接入后） | 已投入 | 最高+实时 | 竞争壁垒 |

## 2. 品类洞察

### 品类规模与趋势

| 需要回答的问题 | 数据来源 | 分析产出 |
|---------------|----------|----------|
| 品类市场规模多大？ | 行业报告 | 「XX品类年规模XXX亿，YYY%增长」 |
| 增长驱动力是什么？ | 趋势分析 | 「增长来自新人群还是消费升级？」 |
| 价格带怎么分布？ | 终端价格监测 | 「超高端XX%/次高端XX%/中端XX%/大众XX%」 |
| 品类生命周期？ | 销量曲线 | 「导入期/成长期/成熟期/衰退期」 |

### 品类生命周期 × 营销策略

| 阶段 | 特征 | 策略重点 | 资源分配 |
|------|------|----------|:--:|
| **导入期** | 品类认知低、增速快但基数小 | 品类教育、建立认知 | 品类教育 50% |
| **成长期** | 认知扩散、新品牌涌入 | 抢份额、建立品牌认知 | 品牌建设 40% + 效果 40% |
| **成熟期** | 增长放缓、格局稳定 | 维护份额、效率竞争 | 效果 60% + 品牌 20% |
| **衰退期** | 销量下滑、品牌退出 | 收割利润、或转型创新 | 维护 80% |

## 3. 消费者洞察

### 消费者画像框架

```
谁（Demographics）：年龄 / 性别 / 城市 / 收入 / 职业
怎么买（Behavior）：渠道 / 频次 / 场景 / 客单价
为什么（Motivation）：JTBD——TA雇佣这个产品完成什么任务
为什么不（Barrier）：尝试障碍 / 购买障碍 / 复购障碍
```

### 酒水消费者决策树

```
什么场景？
  ├─ 自己喝 → 什么心情？
  │   ├─ 放松 → 低度/果酒/精酿
  │   ├─ 犒赏 → 威士忌/高端白酒
  │   └─ 助眠 → 低度/红酒
  ├─ 社交喝 → 和谁？
  │   ├─ 朋友 → 啤酒/预调酒/中端白酒
  │   ├─ 商务 → 高端白酒/洋酒
  │   └─ 约会 → 红酒/香槟
  └─ 送礼 → 什么场合？
      ├─ 过年 → 高端白酒礼盒
      ├─ 探望 → 保健品酒/红酒
      └─ 感谢 → 威士忌/高端白酒
```

### 消费者研究的最小数据集

每季度至少收集一次：

| 数据点 | 获取方式 | 用途 |
|--------|----------|------|
| 品牌无提示知名度 | 问卷/社媒抽样 | 品牌认知趋势 |
| 购买驱动因素排序 | 问卷（「你选这款酒最看重什么」） | 验证定位是否有效 |
| 使用场景分布 | 问卷+终端观察 | 场景策略验证 |
| 竞品转换原因 | 问卷（「你以前喝XX，为什么换」） | 竞争策略输入 |
| NPS | 问卷 | 口碑健康度 |

## 4. 竞争洞察

### 竞品分层

| 层级 | 定义 | 数量 | 追踪频率 | 追踪内容 |
|:--:|------|:--:|:--:|------|
| **核心竞品** | 同价格带+同场景+同人群 | 2-3 | 每周 | 价格/促销/新品/经销商变化 |
| **泛竞品** | 同品类但不同价格带 | 5-8 | 每月 | 战略动向/新品/重大营销 |
| **替代品** | 不同品类但同场景 | 3-5 | 每季度 | 品类趋势/消费者迁移 |

### 竞品追踪模板

```
竞品：[名称]
价格带：[范围]
核心场景：[场景]

本周动态：
- 终端价格变化：[有/无，具体]
- 促销活动：[有/无，具体]
- 新品/新规格：[有/无，具体]
- 经销商变化：[有/无，具体]
- 社媒/投放：[有/无，具体]

评估：[威胁程度 1-5] [需要的应对]
```

## 7. 洞察输出标准

任何洞察分析交付时必须包含：

```
1. 一句话结论（能让CEO记住的）
2. 支撑数据（来源+时效）
3. 对品牌策略的影响（如果这个洞察是真的，我们应该做什么/不做什么）
4. 置信度 + 数据缺口（标注需要进一步验证的部分）
```

**禁止**：只给数据不给解读、只给趋势不给行动、只给结论不给置信度。
