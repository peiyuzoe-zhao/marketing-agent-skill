## A. 战略分析模型

### A1. STP（细分-目标-定位）

```
Segmentation → Targeting → Positioning
（市场细分）   （目标选择）  （品牌定位）

市场上有哪些不同需求的人群？
           ↓
我们应该选择哪个/哪些人群服务？
           ↓
我们在这个人群心智中应该占据什么位置？
```

**细分变量**（B2C 酒水示例）：

| 变量类型 | 示例维度 |
|----------|----------|
| 人口统计 | 年龄、性别、收入、职业 |
| 地理 | 城市层级（一线/二线/三线）、区域（华东/华南）、气候 |
| 心理 | 生活方式（商务社交型 / 独酌享受型 / 派对社交型） |
| 行为 | 消费频次、饮用场景、品牌忠诚度、购买渠道 |
| 消费档次 | 超高端(>800元) / 高端(300-800) / 中端(100-300) / 大众(<100) |

**目标市场选择策略**：
- 单一细分集中化 → 小众品牌（如只做单一麦芽）
- 产品专业化 → 覆盖多人群单一品类
- 市场专业化 → 服务单一类型客户的所有需求
- 选择性专业化 → 选几个不相关细分各自服务
- 全覆盖 → 巨头策略（茅台、五粮液）

### A2. SWOT 分析

```
         对达成目标
         有利            有害
    ┌──────────┬──────────┐
内  │ Strengths │ Weaknesses│
部  │  优势      │   劣势     │
    ├──────────┼──────────┤
外  │ Opportunities│ Threats │
部  │   机会      │   威胁    │
    └──────────┴──────────┘
```

**SWOT 做对的关键**：
- S/W 必须和竞争对手比较，不是绝对判断
- O/T 必须基于外部环境事实，不是愿望清单
- 做完 SWOT 必须输出交叉策略：SO（用优势抓机会）、WO（补劣势抓机会）、ST（用优势避威胁）、WT（防守策略）

### A3. PESTEL 宏观环境分析

| 维度 | 分析内容 | 酒水行业相关变量 |
|------|----------|-----------------|
| Political | 政策、监管、税收 | 白酒消费税、限制三公消费、酒精广告法规 |
| Economic | GDP、通胀、消费力 | 居民可支配收入、消费升级/降级 |
| Social | 人口、文化、价值观 | 年轻一代饮酒习惯变化（低度化、微醺经济） |
| Technological | 技术变革 | 电商渗透率、AI 营销、区块链防伪 |
| Environmental | 环保、可持续 | 酿酒排放、包装环保法规 |
| Legal | 法律框架 | 酒类专卖、未成年禁售、广告法 |

### A4. Porter 五力分析

```
            潜在进入者威胁
                  ↓
供应商议价能力 → 行业竞争强度 ← 买方议价能力
                  ↑
            替代品威胁
```

| 五力 | 酒水行业典型状况 |
|------|-----------------|
| 行业内竞争 | 品牌竞争（非价格竞争为主）、区域割据 |
| 潜在进入者 | 新品牌需要渠道+品牌积累，门槛中高 |
| 供应商议价 | 基酒/原料/包材供应商，集中度低则议价弱 |
| 买方议价 | 品牌方 vs 经销商/终端，渠道话语权博弈 |
| 替代品 | 不同酒种之间（白酒 vs 威士忌 vs 低度酒） |

### A5. BCG 矩阵（产品组合管理）

```
市场增长率
    ↑
 高 │  ★ Question Marks   │  ★ Stars
    │  问题儿童             │   明星
    │  （低份额+高增长）    │  （高份额+高增长）
    │  决策：选择性投入      │  决策：持续投资
    ├───────────────────────┤
 低 │  × Dogs              │  $ Cash Cows
    │  瘦狗                 │   现金牛
    │  （低份额+低增长）    │  （高份额+低增长）
    │  决策：收割/退出      │  决策：维持、收割利润
    └───────────────────────┘
      低 ← 相对市场份额 → 高
```

### A6. Ansoff 增长矩阵

```
             现有市场      新市场
          ┌─────────┬─────────┐
现有产品  │ 市场渗透 │ 市场开发 │
          │（最安全）│（新渠道  │
          │         │  /新区域）│
          ├─────────┼─────────┤
新产品    │ 产品开发 │ 多元化   │
          │（新产品线│（最危险）  │
          │ /新品类）│          │
          └─────────┴─────────┘
```

**酒水应用**：
- 市场渗透：同一款酒铺更多终端、加密区域覆盖
- 市场开发：区域酒企跨省、白酒品牌出海
- 产品开发：传统白酒推低度版、果酒线
- 多元化：酒企跨界饮料、文旅

## B. 消费者/用户模型

### B1. AIDA

```
Attention → Interest → Desire → Action
  注意         兴趣        渴望      行动
```

所有广告效果的祖父级模型。简单、线性、清晰。但过于简化——假设消费者被动接收信息然后线性决策，现代消费者行为远比这复杂。

### B2. 消费者决策旅程 CDJ / 5A（Kotler）

```
 Aware → Appeal → Ask → Act → Advocate
 了解     吸引     问询   购买     拥护
  ↓                               ↑
  └───────────────────────────────┘
            循环、非线性的
```

**5A 各阶段营销任务**：

| 阶段 | 营销任务 | KPI |
|------|----------|-----|
| Aware | 让消费者「知道有我」 | 曝光量、触达率 |
| Appeal | 让消费者「喜欢我」 | 品牌偏好度、NPS |
| Ask | 消费者主动搜索时「找到我」 | 搜索量、内容覆盖 |
| Act | 购买便利、体验好 | 转化率、复购率 |
| Advocate | 推荐给他人 | 推荐率、UGC 量 |

### B3. Jobs to be Done (JTBD)

核心洞察：消费者不是买产品，是「雇佣」产品来完成某件「任务」。
关键是理解 **「消费者在什么场景下、要完成什么进度」**。

**JTBD 句式**：
> 当我在 [场景] ，我想 [完成什么] ，以便 [获得什么结果] 。

**酒水 JTBD 示例**：
- 「当我在商务宴请时，我想点一款客人觉得有面子的酒，以便促成合作」
- 「当我周末独处时，我想喝点微醺放松一下，以便缓解一周的压力」
- 「当我和朋友聚会时，我想喝点好喝不贵的，以便热闹而不破费」

### B4. RFM 模型（客户分层）

```
R (Recency) : 最近一次消费距今多久？
F (Frequency): 消费频次？
M (Monetary) : 消费金额？
```

每个维度分 1-5 分（或高/中/低三段），组合后形成客户分层：
- 重要价值客户 (R高F高M高) → 保留
- 重要发展客户 (R高F低M高) → 激活
- 重要保持客户 (R低F高M高) → 挽回
- 一般客户 → 低优先级

## C. 品牌传播模型

### C1. DAGMAR（定义广告目标以衡量广告效果）

核心：广告目标必须**具体、可衡量**。不是「提升品牌知名度」，而是「在目标人群中，将品牌无提示知名度从 30% 提升到 50%，6 个月内」。

**四阶段传播效果**：
```
认知 (Awareness) → 理解 (Comprehension) → 信服 (Conviction) → 行动 (Action)
```

### C2. FCB Grid（创意策略象限）

```
           高卷入             低卷入
        ┌──────────┬──────────┐
 理性   │  Informative│  Habitual  │
        │  Learn-Feel-Do│ Do-Learn-Feel│
        │  车、房、家电│  日用品、调味品│
        ├──────────┼──────────┤
 感性   │  Affective   │  Satisfaction│
        │  Feel-Learn-Do│ Do-Feel-Learn│
        │  珠宝、香水  │  啤酒、零食  │
        └──────────┴──────────┘
```

**酒水典型落位**：
- 高端白酒/威士忌 → 感性 + 高卷入 (Affective)
- 日常啤酒 → 感性 + 低卷入 (Satisfaction)
- 投资级红酒 → 理性 + 高卷入 (Informative)

### C3. 6Ms 传播规划框架

| M | 含义 | 关键问题 |
|----|------|----------|
| Mission | 传播目标 | 要达成什么？认知/态度/行为？ |
| Market | 目标受众 | 对谁说？ |
| Money | 预算 | 花多少钱？怎么分配？ |
| Message | 传播信息 | 说什么？ |
| Media | 媒体/渠道 | 在哪说？ |
| Measurement | 效果衡量 | 怎么知道做成了？ |

## D. 数字化/增长模型

### D1. AARRR（Pirate Metrics 海盗指标）

```
Acquisition  →  Activation  →  Retention  →  Revenue  →  Referral
   获客            激活           留存          变现        推荐
```

| 阶段 | 关键指标 | 酒水 DTC 示例 |
|------|----------|-------------|
| Acquisition | CAC（获客成本）、渠道来源占比 | 抖音信息流获客成本 |
| Activation | 首次购买率、注册完成率 | 首单转化率 |
| Retention | 复购率、流失率 | 季度复购率 |
| Revenue | ARPU、LTV | 客单价、年消费额 |
| Referral | K因子（病毒系数） | 分享率、裂变订单占比 |

### D2. RACE 框架

```
Reach → Act → Convert → Engage
触达     互动     转化      维系
```

侧重数字营销全流程，比 AARRR 更适合品牌电商场景。

### D3. 增长飞轮

```
        获客
       ↗   ↘
    推荐     激活
      ↖   ↙
    变现 ← 留存
```

与线性漏斗不同，飞轮强调循环加速——每个环节的改善都让下一圈转得更快。关键：找到飞轮的「第一推动力」（通常是产品体验或内容）。

## E. 竞争动态应对

Porter 5 Forces 是静态分析。竞争对手不会等你分析完再动。本节覆盖竞争信号的识别、分级与响应。

### 竞争信号识别

| 竞品动作 | 是威胁吗？ | 判断标准 |
|----------|:--:|------|
| 终端降价 | **是**——如果发生在你的核心市场 | 降价幅度 > 10% 且持续 2 周以上 = 战略性降价，不是促销 |
| 新品上市 | **可能是**——如果和你的价格带重叠 | 价格带重叠 + 场景重叠 + 区域重叠 = 直接威胁 |
| 挖你的经销商 | **是**——信号比实际损失更重要 | 一个经销商被挖 = 所有经销商在观望你的反应 |
| 加大广告投放 | **不一定**——可能是品类教育，对你有利 | 看投放内容是否直接针对你的核心场景/人群 |
| 媒体负面稿 | **看内容** | 行业普遍问题→不跟；针对性攻击→必须回应 |

### 竞争响应矩阵

| 威胁等级 | 定义 | 响应速度 | 响应策略 |
|:--:|------|:--:|------|
| **L4 战略级** | 竞品用更低价格进入你的核心市场，持续 2 周以上 | < 72h | 必须跟——但不要跟价格，跟价值（品鉴会、终端促销员、消费者扫码红包） |
| **L3 战术级** | 竞品在局部区域降价促销 | < 1 周 | 局部跟进促销，但不降标价。守住 A 类终端 |
| **L2 信号级** | 竞品发布新品、签约新经销商 | 监测 | 不跟，但做应对预案——如果 3 个月后这个新品在你核心市场铺货了，你怎么办 |
| **L1 噪音级** | 竞品日常广告投放、社媒活动 | 忽略 | 不跟。大多数竞争动作只是噪音 |

### 响应铁律

1. **不跟价格战**。除非你的供应链成本结构竞品无法复制。
2. **跟价值不跟价格**。竞品降价→你加品鉴会、加促销员、加扫码红包。守住终端推荐权比守住标价更重要。
3. **在竞品还没站稳的市场上反向操作**。竞品打你华东→你去打他华南——让他双线作战。
4. **经销商信心比消费者感知更重要**。竞品动作的第一冲击波是经销商心理战——第一时间安抚经销商比降价比什么都管用。

### 竞争情报体系

| 情报来源 | 获取方式 | 频率 |
|----------|----------|:--:|
| 终端走访（业务代表） | 拜访时记录竞品陈列/价格/促销 | 每周 |
| 经销商反馈 | 经销商主动告知（他们同时做竞品的代理） | 自然流入 |
| 电商价格监控 | 爬虫或手动检查竞品旗舰店价格 | 每周 |
| 社媒监听 | 竞品官号、行业 KOL 提及竞品的内容 | 持续 |
| 糖酒会/行业活动 | 竞品展位、新品发布 | 按行业节奏 |

## F. 品牌组合资源分配

BCG Matrix 告诉你什么是明星什么是金牛——但没告诉你**怎么分钱**。

### 70-20-10 之外的实际分配逻辑

| 产品类型 | 核心目标 | 预算占比 | 判断标准 |
|------|------|:--:|------|
| **明星**（高增长+高份额） | 加速增长，建立壁垒 | 40-50% | ROAS > 3 或有战略价值 |
| **现金牛**（低增长+高份额） | 维持，收割利润 | 15-20% | 利润贡献率——不是不投，是精准投 |
| **问题儿童**（高增长+低份额） | 验证，选择性投入 | 20-30% | 3-6 个月验证期——验证不过则砍 |
| **瘦狗**（低增长+低份额） | 最小化投入或退出 | <5% | 除非有战略防御价值，否则不投 |

### 分配决策的四问

季度预算分配前，对每条产品线问：

1. **如果一刀砍掉这条产品线，品牌受伤多大？** ← 防御价值
2. **如果加倍投入，能看到什么结果？** ← 增长潜力
3. **这条线现在的利润，够它自己的推广费吗？** ← 自造血能力
4. **它和主品牌的关联度？帮主品牌还是吃主品牌？** ← 协同价值

### 新品 vs 老品分配

```
品牌总预算 100%

成熟产品线（维护+收割）：30-40%
  └── 现金牛：维持终端、防竞品偷袭
新品线（增长）：40-50%
  └── 明星/问题儿童：最大份额给最有确定性增长的
品类教育/品牌建设：10-20%
  └── 无论新品老品，品类的护城河要持续挖
实验：5-10%
  └── 新渠道、新场景、新人群——永远留实验预算
```

### 区域分配

| 区域类型 | 特征 | 预算策略 |
|------|------|------|
| **核心市场** | 份额高、利润高 | 维护型投入——防竞品、保利润 |
| **增长市场** | 份额中、增速快 | 进攻型投入——抢份额，预算倾斜 |
| **潜力市场** | 份额低、增速快 | 试探型投入——小规模验证 |
| **观望市场** | 份额低、增速慢 | 最小维护——不主动投入 |

## G. 度量与归因

品牌方最痛的问题：「花了 500 万，品牌搜索涨了 30%——投流、KOL、经销商铺货，各自贡献了多少？」归因回答的是**哪个动作带来了增长**。

### 归因的三种方法

| 方法 | 原理 | 优点 | 缺点 | 适用 |
|------|------|------|------|------|
| **最后点击归因** | 转化前的最后一次点击拿全部功劳 | 简单、可执行 | 严重低估上层漏斗（品牌广告、种草） | 纯效果广告 |
| **线性归因** | 转化路径上每个触点平均分 | 比最后点击公平 | 不区分触点的真实影响力 | 触点少的场景 |
| **营销组合建模 (MMM)** | 统计模型——把销量拆解到各渠道 | 最科学、涵盖线上+线下 | 需要至少 1 年数据、专业团队 | 有数据积累的品牌 |

**实战建议**：初创品牌用最后点击归因就够了（先跑起来）。年营收过亿后，引入 MMM。中间阶段用「渠道分组对比」——两个相似城市做不同策略组合，看 3 个月后的差异。

### 归因窗口

| 品类 | 建议归因窗口 | 原因 |
|------|:--:|------|
| 快消品/冲动购买 | 1-3 天 | 决策周期短 |
| 酒水（啤酒/低度酒） | 1-7 天 | 冲动+即时消费 |
| 酒水（中高端白酒/威士忌） | 7-30 天 | 涉及比价/送礼决策/种草到购买 |
| 酒水（收藏级/礼品） | 30-90 天 | 决策周期长 |

### Geo-test 实验设计

在两个高度相似的城市，做不同策略组合，3 个月后比较。

```
城市 A（实验组）：投流+KOL+经销商铺货，预算 万元
城市 B（对照组）：只做经销商铺货，预算 万元

3 个月后比较：
- 销量差异 = 投流+KOL 的增量贡献
- 品牌搜索差异 = 上层漏斗贡献
- 经销商进货频率差异 = 渠道信心贡献
```

**选城市标准**：人口规模相似、消费水平相似、品牌基础相似（都铺货 3 个月以上）、不相邻（避免窜货干扰）。

### 品牌增长归因框架

品牌增长 = 品类增长 + 市场份额增长。先分清楚是「水涨船高」还是「你真的更好了」。

```
品类是否在增长？
  ├─ 是 → 你的增速是否 > 品类增速？
  │   ├─ 是 → 你在抢份额 ✓
  │   └─ 否 → 你被竞品抢份额 ✗
  └─ 否 → 品类在下滑
      ├─ 你持平 → 你在维稳
      └─ 你也下滑 → 双重压力
```
