# Offline Retail — 线下门店营销

> 加载时机：终端门店策略、店内营销设计、陈列管理、终端动销诊断、门店分级管理。

---

## 1. 终端门店的核心地位

对于酒水行业，线下终端是**不可替代的战场**：
- 酒水消费高度场景化（餐饮、夜店、聚会），即时性消费只能在终端发生
- 终端是品牌体验的物理载体——瓶型、包装、冰镇温度、倒酒仪式，线上无法传递
- **终端动销 = 品牌健康的真实体温计**——线上数据可以刷，终端开瓶率骗不了人

```
品牌 → 经销商 → 终端门店 → 消费者
                    ↑
              「最后一公里」
              货架份额 = 心智份额的物理映射
```

## 2. 酒水终端门店类型全景

### 2.1 即饮渠道（On-Trade）

| 终端类型 | 特征 | 酒水消费特点 | 营销切入点 |
|----------|------|-------------|-----------|
| **夜店/Club** | 高溢价、高消费、潮流导向 | 洋酒/香槟/高端啤酒为主，开瓶率极高 | 品牌驻场、包房冠名、专属调酒 |
| **酒吧/清吧** | 中高消费、品质导向 | 威士忌/精酿/鸡尾酒基酒 | 品鉴合作、调酒师培训、品牌之夜 |
| **KTV** | 量贩式消费、社交导向 | 啤酒/预调酒/中档洋酒 | 套餐绑定、包厢陈列 |
| **餐饮（中高端）** | 宴请/聚餐场景 | 白酒/红酒为主 | 酒单推荐、开瓶奖励、品鉴赠饮 |
| **餐饮（大众）** | 高频日常消费 | 啤酒/光瓶白酒 | 冰柜陈列、套餐搭售 |

### 2.2 非即饮渠道（Off-Trade）

| 终端类型 | 特征 | 营销切入点 |
|----------|------|-----------|
| **烟酒店** | 传统白酒主战场，熟人经济 | 陈列费、推头费、老板佣金 |
| **商超/大卖场** | 大众消费、品牌曝光 | 端架陈列、促销堆头、试饮 |
| **便利店** | 即时消费、年轻人群 | 冰柜抢占、小瓶装铺货 |
| **专卖店/体验店** | 品牌形象、深度体验 | 品鉴区、品牌文化墙、会员体系 |
| **精品酒窖/买手店** | 高端小众、专业推荐 | 品鉴活动、限量发售 |

### 2.3 终端特征对比

| 维度 | 即饮（On-Trade） | 非即饮（Off-Trade） |
|------|:--:|:--:|
| 消费决策 | 场景驱动，即时决策 | 计划性购买，受陈列/促销影响大 |
| 溢价能力 | 高（2-5倍加价） | 低-中 |
| 品牌体验深度 | 深（完整消费体验） | 浅（货架接触） |
| 营销可控性 | 低（依赖终端配合） | 中-高（陈列/促销可控） |
| 动销数据可得性 | 低 | 中（POS数据） |

## 3. 终端门店分级管理

### 3.1 A/B/C 分级体系

| 级别 | 定义 | 占比 | 销量贡献 | 管理策略 |
|:--:|------|:--:|:--:|------|
| **A 类** | 高销量/高势能门店 | 10-15% | 50-60% | 专人维护、周访、优先资源 |
| **B 类** | 中等销量/有增长潜力 | 25-30% | 25-30% | 定期拜访、标准陈列、促销支持 |
| **C 类** | 低销量/长尾覆盖 | 55-65% | 10-15% | 按需拜访、最低陈列标准 |

### 3.2 分级指标参考

```
评分 = 
  月均销量 × 0.4 +
  门店类型势能 × 0.2 +
  陈列位置质量 × 0.15 +
  店主配合度 × 0.1 +
  竞品排斥度 × 0.1 +
  区域辐射影响力 × 0.05
```

### 3.3 门店生命周期管理

```
新开门店 → 铺货期（1-2月）→ 成长期（2-6月）
                            ├→ 升级为 A/B 类 → 维护期
                            └→ 持续低迷 → 评估淘汰
```

## 4. 店内营销 (In-Store Marketing)

### 4.1 陈列策略

**陈列黄金法则**：

| 法则 | 内容 |
|------|------|
| **视线高度** | 货架 1.2-1.6 米 = 黄金陈列位 |
| **右手法则** | 消费者视线自然偏右，优先抢占右侧 |
| **品牌集中** | 同一品牌产品集中陈列，形成视觉冲击 |
| **先进先出** | 旧货前置，新货后置，防止临期 |
| **饱满陈列** | 不满的货架 = 没人买的信号 |

**特殊陈列形式**：

| 形式 | 效果 | 适用终端 |
|------|------|----------|
| 端架/端头 | 独立展示，溢价最高 | 商超、便利店 |
| 堆头/地堆 | 量大冲击，适合促销 | 商超、烟酒店 |
| 冰柜专属层 | 冰镇即饮场景 | 便利店、餐饮 |
| 背柜展示 | 高端形象 | 烟酒店、专卖店 |
| 吧台陈列 | 直接面对消费决策 | 酒吧、KTV |

### 4.2 POP 物料体系

| 物料 | 目的 | 关键设计原则 |
|------|------|-------------|
| 价格签/跳跳卡 | 传递促销信息 | 大字号、高对比色、限时感 |
| 海报/立牌 | 品牌形象+活动告知 | 视觉焦点、一句话信息 |
| 桌卡/台卡 | 餐饮场景直接触达 | 双面设计（消费者面+服务员面） |
| 灯箱/发光字 | 店外引流 | 夜间可视性 |
| 试饮台 | 降低尝试门槛 | 整洁、专业、有品牌调性 |

### 4.3 店内促销

**促销类型决策**：

| 促销方式 | 优点 | 缺点 | 适用场景 |
|----------|------|------|----------|
| 直接降价 | 拉动效果最强 | 伤害品牌、价格锚点下移 | 清库存 |
| 买赠（买一送一/买大送小） | 价值感强，不伤害标价 | 可能培养消费者等促销 | 新品推广、节假日 |
| 满减（满 N 减 M） | 提升客单价 | 门槛设置影响转化 | 烟酒店/商超 |
| 开瓶有奖/扫码红包 | 直接激励消费 | 依赖消费者主动扫码 | 即饮渠道 |
| 套餐搭售 | 连带销售、去库存 | 搭配不当时反感 | 餐饮、夜店 |
| 试饮/赠饮 | 零门槛体验 | 成本高、转化不确定 | 新品入市 |

## 5. 终端动销管理

### 5.1 核心指标

| 指标 | 公式 | 含义 | 警戒信号 |
|------|------|------|----------|
| **铺货率** | 有货门店/目标门店 | 覆盖广度 | < 60% = 渠道太弱 |
| **排面份额** | 品牌排面数/品类总排面 | 店内竞争力 | 远低于销量份额 = 有流失风险 |
| **库存周转天数** | 平均库存/(日均销量) | 资金效率 | > 90 天 = 库存积压 |
| **缺货率** | 缺货SKU/总SKU | 供应链健康 | > 5% = 丢失销售机会 |
| **开瓶率** | 实际消费/进货量 | 真实动销（即饮渠道） | 低 = 货压在终端没卖掉 |
| **复购率** | 二次购买门店/总铺货门店 | 产品力 | < 30% = 产品不被接受 |

### 5.2 动销诊断模型

```
问题：门店不动销
        ↓
第1层排查：有没有人看得到？
  → 铺货了吗？陈列位置好吗？POP物料到位吗？
        ↓
第2层排查：看到了为什么没买？
  → 价格合理吗？包装吸引人吗？有促销吗？
        ↓
第3层排查：买了为什么没再买？
  → 好喝吗？价格值吗？竞品更强吗？
        ↓
第4层排查：消费者根本没进店？
  → 门店客流够吗？品类本身有需求吗？
```

| 角色 | 核心职责 | 考核指标 | 管理要点 |
|------|----------|----------|----------|
| **业务代表** | 拜访、补货、陈列维护、客情 | 拜访达成率、铺货率、回款 | 路线规划、拜访SOP |
| **促销员/BA** | 推荐、试饮、促进即时购买 | 当班销量、试饮转化率 | 排班、话术培训、激励 |

### 6.2 促销员话术框架（FABE）

```
F (Feature)    → 属性：这款酒用的是XX产区XX年份的葡萄
A (Advantage)  → 优势：它的单宁比同价位的更柔顺
B (Benefit)    → 利益：所以入口不涩，女士也能轻松喝
E (Evidence)   → 证据：它在XX盲品会上拿了金奖
```

## 7. 门店数据采集体系

### 7.1 最小数据采集集

每次终端拜访，至少采集：

| 数据项 | 采集方式 | 用途 |
|--------|----------|------|
| 库存量（分SKU） | 人工盘点 | 计算周转、预警缺货/积压 |

| 陈列照片 | 手机拍照 | 陈列合规检查 |
| 店主反馈 | 面对面 | 定性洞察（竞品动作、消费者反馈） |
| 销量（如有POS） | 系统导出 | 动销分析 |

### 7.2 数字化工具

- **SFA（销售自动化）**：业务代表拜访路线、打卡、陈列拍照上传
- **DMS（经销商管理系统）**：进销存数据
- **终端 POS 数据直连**：实时动销（理想状态，只有 A 类终端可做到）

## 渠道谈判与贸易政策

品牌方和经销商之间不是「我帮你做陈列」，是持续的商业谈判。品牌想做品牌，经销商想做周转——中间的摩擦是日常。

### 经销商利益结构

经销商只关心三件事，顺序固定：

```
1. 利润 → 「我卖一瓶赚多少钱？」（利润率 × 周转速度）
2. 风险 → 「压货了卖不掉怎么办？」（库存积压 = 现金被锁死）
3. 关系 → 「跟你合作舒服吗？」（客情、服务、响应速度）
```

品牌方犯的最大错误：跟经销商讲品牌故事，但经销商只关心第一条。

### 经销商利润模型

```
经销商年利润 = 单品毛利 × 年销量 — 库存资金成本 — 配送成本 — 返利条件风险

单品毛利 = (一批价 — 出厂价) + 返利 + 其他激励
库存资金成本 = 平均库存量 × 出厂价 × 资金利率 × 周转天数/365
```

**关键洞见**：经销商利润不只看价差。一个价差 25% 但周转慢的产品，不如价差 15% 但周转快的产品。周转 > 价差。

### 经销商政策设计

| 政策要素 | 设计原则 | 常见陷阱 |
|----------|----------|----------|
| **基础 margin** | 不低于同品类竞品，保持竞争力 | 单纯比 margin 是价格战 |
| **季度返利** | 和销量目标挂钩，阶梯式 | 目标不可达成→返利形同虚设 |
| **年度返利** | 和全年总目标挂钩，金额要大才有效 | 金额太小→经销商无感 |
| **新品铺货激励** | 首批进货折扣+陈列支持 | 激励结束后终端不动销→库存积压 |
| **账期** | 新经销商现金，老经销商月结 30-60 天 | 账期一开，永无回头 |

### 返利阶梯设计

```
季度返利阶梯（示例）：
  完成 80%  → 返利 1%
  完成 100% → 返利 2%
  完成 120% → 返利 3%
  完成 150% → 返利 4%（+ 额外奖励）

关键：目标的 80% 必须可达，否则经销商从一开始就放弃。
```

### 控价策略

| 策略 | 怎么做 | 适用 |
|------|--------|------|
| **最低终端价** | 合同中约定终端不得低于 XX 元，违规扣返利 | 品牌力强的品牌 |
| **建议零售价** | 给建议、不强控，通过价差设计自然引导 | 品牌力中等 |
| **窜货处罚** | 跨区域销售的经销商扣除保证金/返利 | 所有品牌 |

### 经销商谈判准备

你去找经销商谈判前，必须回答经销商心里的三个问题——即使他不问出口：

1. **「我为什么要做你的品牌？」** ← 利润？品牌背书？你帮我做终端？
2. **「我放弃现有品牌做你，风险是什么？」** ← 动了谁的蛋糕？终端接受吗？
3. **「如果卖不动，你有什么后备方案？」** ← 换货政策？退货政策？促销支持？

### 谈判工具包

- 给经销商看的**一页纸**：不是品牌手册，是「做我们这个品牌，你一年能赚多少钱」
- **试销方案**：不要求经销商一步到位压满货——「先拿 50 箱试 3 个月，跑通了再扩大」
- **竞品对比表**：不是贬低竞品，是客观比较——「同价位，我们的利润率比他们高 5 个点」
